Loops

basic for-loops, 1. variable initialization, 2. condition checking, 3. block execution and 4. after block execution. we did all that with while-loops too, just looks different 

for (let counter = 0; counter <= 6; counter += 1) {
  console.log(counter);
}
// -> 0, 1, ..., 6
as usual there are fancy things you can do but normally don't, like multiple variables involved 

for (let counter = 0, moreCounter = 0; counter <= 6; counter += 5, moreCounter++) {
  console.log(counter, moreCounter);
}
// -> 0 0
// -> 5 1
