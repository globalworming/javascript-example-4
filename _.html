Arrays

Arrays can act as list, stack or queue. Set and get values with the brackets notation 

const someList = ["a", "b", "c"]
console.log(someList[0]) // -> "a"
console.log(someList[2]) // -> "c"
console.log(someList[2 - 1]) // -> "b"
Arrays are special kinds of objects 

console.log(typeOf []) // -> "object"
All these have shorthands for calculation and assignment 

const kickDamage = 10
let health = 100
health -= kickdamage // same as health = health - kickdamage
console.log(health) // -> 90
