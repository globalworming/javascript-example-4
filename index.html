Loops

Repeat things a lot, with variants how. While loops are the simplest, just repeat something as long as a provided condition is true 

while (true) { // endless loop, don't execute :)
    console.log("say hi")
}
the condition is checked first, if true, the block is executed 

let courses = 0
while (courses < 10) {
    console.log("subscribe")
    courses++
}
you can break out of loops early 

let courses = 0
while (true) {
    courses++
    break
}
console.log(courses) // -> 1
