Loops

there are a lot more loops like 

for (;;) {
    console.log("i would loop endlessly");
}
that's the for loop from before but without the initialization, condtion and final-expression. these are all optional actually.

then there are loops specifically for objects (for-in) and arrays (for-of), so let's talk about arrays and objects soon. 


const array = ['a', 1, true];
for (const element of array) {
  console.log(element);
}
// -> 'a'
// -> 1
// -> true

also this array function acts as loop

const array = ['a', 1, true];
array.forEach(element => console.log(element))
// -> 'a'
// -> 1
// -> true


const object = { a: 1, b: 2, c: 3 };
for (const property in object) {
  console.log(`${property}: ${object[property]}`);
}
// "a: 1"
// "b: 2"
// "c: 3"
