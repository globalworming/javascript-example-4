Conditional

for when you sometimes want to do different things. the "if" statement with a condition is used to create execution branches 

if (7 < 13) console.log("7 is less than 13")
often we need to distinguish between an expression being true or false, the "else" statement is used here 

if (Math.random() < 0.5) console.log("random number less 0.5")
else console.log("greater or equal 0.5")
multiple statements need to go in a block 

if (Math.random() < 0.5) {
    console.log("true")
    console.log("random number less than 0.5")
} else {
    console.log("false")
    console.log("greater or equal 0.5")
}
multiple conditions can be chained need to go in a block 

const randomNumber = Math.random()

if (randomNumber < 0.5) {
    console.log("random number less than 0.5")
} else if (randomNumber < 0.9) {
    console.log("less than 0.9")
} else {
    console.log("greater or equal 0.9")
}
basic if-else condition can be replaced with the ternary operator "?:" 

if (Math.random() < 0.5) console.log("random number less 0.5")
else console.log("greater or equal 0.5")

// same as
Math.random() < 0.5 ? console.log("random number less 0.5") : console.log("greater or equal 0.5")

// and might be used as conditional assignment
const message = Math.random() < 0.5 ? "smaller" : "bigger"
checking for undefined, null or other "falsy" things can be replaced with the "optional chaining" operator "?." 

const array = ["text"]
let result;
if (array[0]) result = array[0].toUpperCase()
// same as
const array = ["text"]
let result = array[0]?.toUpperCase()
